## 二、A 系统接口详细说明

### 2.1 获取企业访问令牌

**接口路径**: `POST https://open.fxiaoke.com/cgi/corpAccessToken/get/V2`

**请求体**:

| 字段名           | 类型     |  必填 | 说明    |
| ------------- | ------ | :-: | ----- |
| appId         | string |  是  | 应用ID  |
| permanentCode | string |  是  | 永久授权码 |
| appSecret     | string |  是  | 应用密钥  |

**请求示例**:

```json
{
  "appId": "FSAID_1324417",
  "permanentCode": "C13DD771E81BD8AC8CC8DB5525DBBAD1",
  "appSecret": "fd2587a581354fc0836333029ed1d1cc"
}
```

**响应体**:

| 字段名             | 类型      | 说明         |
| --------------- | ------- | ---------- |
| corpAccessToken | string  | 企业访问令牌     |
| corpId          | string  | 企业ID       |
| expiresIn       | integer | 有效期（秒）     |
| errorCode       | integer | 错误码（0表示成功） |
| errorMessage    | string  | 错误信息       |

**响应示例**:

```json
{
  "corpAccessToken": "8EA0191CCFAF03F1D6904CEDC3A6832B",
  "corpId": "FSCID_7093D2ACD92F463794F703908F1BA9EB",
  "expiresIn": 7200,
  "errorCode": 0,
  "errorMessage": "success"
}
```

**测试结果**: ✅ 成功

***

### 2.3 查询客户数据

**接口路径**: `POST https://open.fxiaoke.com/cgi/crm/v2/data/query`

**请求体**:

| 字段名                                      | 类型      |  必填 | 说明               |
| ---------------------------------------- | ------- | :-: | ---------------- |
| corpAccessToken                          | string  |  是  | 企业访问令牌           |
| currentOpenUserId                        | string  |  是  | 当前用户ID           |
| corpId                                   | string  |  是  | 企业ID             |
| data                                     | object  |  是  | 查询参数             |
| data.dataObjectApiName                   | string  |  是  | 对象名称（AccountObj） |
| data.find\_explicit\_total\_num          | boolean |  是  | 是否返回总数           |
| data.search\_query\_info                 | object  |  是  | 查询条件             |
| data.search\_query\_info.offset          | integer |  是  | 偏移量              |
| data.search\_query\_info.limit           | integer |  是  | 每页数量             |
| data.search\_query\_info.orders          | array   |  否  | 排序条件             |
| data.search\_query\_info.fieldProjection | array   |  是  | 返回字段列表           |
| data.search\_query\_info.filters         | array   |  否  | 过滤条件             |

**请求示例**:

```json
{
  "corpAccessToken": "8EA0191CCFAF03F1D6904CEDC3A6832B",
  "currentOpenUserId": "FSUID_4AA04675CFE7E17361175F89E9BDF92A",
  "corpId": "FSCID_7093D2ACD92F463794F703908F1BA9EB",
  "data": {
    "find_explicit_total_num": true,
    "search_query_info": {
      "offset": 0,
      "limit": 100,
      "orders": [
        {
          "fieldName": "create_time",
          "isAsc": true
        }
      ],
      "fieldProjection": ["name", "owner", "account_level", "address", "data_own_department"],
      "filters": [
        {
          "operator": "GT",
          "field_name": "create_time",
          "field_values": [1779379200000]
        }
      ]
    },
    "dataObjectApiName": "AccountObj"
  }
}
```

**响应体**:

| 字段名                                    | 类型      | 说明         |
| -------------------------------------- | ------- | ---------- |
| data                                   | object  | 数据对象       |
| data.dataList                          | array   | 客户列表       |
| data.dataList\[].\_id                  | string  | 客户唯一标识     |
| data.dataList\[].name                  | string  | 公司名称       |
| data.dataList\[].address               | string  | 公司地址       |
| data.dataList\[].account\_level        | string  | 客户级别（1-6）  |
| data.dataList\[].owner                 | array   | 负责人ID列表    |
| data.dataList\[].data\_own\_department | array   | 所属部门ID列表   |
| data.offset                            | integer | 当前偏移量      |
| data.limit                             | integer | 每页数量       |
| data.total                             | integer | 总记录数       |
| errorCode                              | integer | 错误码（0表示成功） |
| errorMessage                           | string  | 错误信息       |

**响应示例**:

```json
{
  "data": {
    "dataList": [
      {
        "_id": "6246b4bbf103b2000152ba42",
        "name": "云南云电同方科技有限公司",
        "address": "云南省昆明市高新技术开发区西二环路中段398号",
        "account_level": "5",
        "owner": ["FSUID_E3D1330CB5F643589D229138BBCF57D2"],
        "data_own_department": ["1006"]
      }
    ],
    "offset": 0,
    "limit": 100,
    "total": 2218
  },
  "errorCode": 0,
  "errorMessage": "OK"
}
```

**测试结果**: ✅ 成功

## 三、B 系统接口详细说明

### 3.1 创建公司

**接口路径**: `POST https://www.bangwo8.com/api/v1/companies.json`

**请求头**:

| 字段名           | 类型     |  必填 | 说明               |
| ------------- | ------ | :-: | ---------------- |
| Authorization | string |  是  | Basic 认证         |
| Content-Type  | string |  是  | application/json |
| Cookie        | string |  是  | 会话Cookie         |

**请求体**:

| 字段名                             | 类型     |  必填 | 说明    |
| ------------------------------- | ------ | :-: | ----- |
| company                         | object |  是  | 公司对象  |
| company.companyName             | string |  是  | 公司名称  |
| company.address                 | string |  是  | 公司地址  |
| company.custom\_fields          | array  |  是  | 自定义字段 |
| company.custom\_fields\[].key   | string |  是  | 字段键名  |
| company.custom\_fields\[].value | string |  是  | 字段值   |

**请求示例**:

```json
{
  "company": {
    "companyName": "云南云电同方科技有限公司",
    "address": "云南省昆明市高新技术开发区西二环路中段398号",
    "custom_fields": [
      {
        "key": "field_1",
        "value": "一般客户（B）"
      }
    ]
  }
}
```

**响应体（成功）**:

| 字段名         | 类型     | 说明   |
| ----------- | ------ | ---- |
| company     | object | 公司对象 |
| company.uId | string | 公司ID |

**响应体（已存在）**:

| 字段名            | 类型     | 说明               |
| -------------- | ------ | ---------------- |
| errorMsg       | string | 错误信息             |
| status         | string | 状态码（110026表示已存在） |
| data.companyId | string | 已存在的公司ID         |

**测试结果**: ✅ 成功（支持创建和已存在处理）
